# ⌨️ Typing Speed Tester

A simple Python program that measures how fast you can type a given sentence.

## 🚀 Features
- Measures typing speed in seconds
- Checks if the sentence matches exactly
- Simple and beginner-friendly

## 📦 Requirements
- Python 3.x
- No external libraries needed

## ▶️ How to Run
python typing_speed.py

# 📸 Sample Output

```
👉 Type the following sentence exactly as shown:

'python is super fun'

⏳ Press Enter when you are ready to start...
✍️ Type here: python is super fun

✅ Great! Time taken: 3.45 seconds
```

# 📝 How It Works
1. The program shows a sentence to type
2. It starts a timer when you press Enter
3. You type the sentence
4. It stops the timer and checks your input
5. It shows your time or asks you to try again
